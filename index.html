Luas Segitiga

Rumus Luas Segitiga yaitu :

L = 1/2 x a x t

Dimana :

L = Luas

a = Panjang Alas

t = Tinggi

Hitung Luas Segitiga

Keliling Segitiga

Rumus Keliling Segitiga yaitu :

K = S1 + S2 + S3

Dimana :

K= Keliling

S1 = AB = Panjang sisi dari titik A ke B

S2 = BC = Panjang sisi dari titik B ke C

S3 = AC = Panjang sisi dari titik A ke C

Hitung Keliling Segitiga

Luas Jajar Genjang

Rumus Luas Jajar Genjang yaitu :

L = a x t

Dimana :

L = Luas

a = Panjang Alas

t = Tinggi

Hitung Luas Jajar Genjang

Keliling Jajar Genjang

Rumus Keliling Jajar Genjang yaitu :

K = 2 * ( a + t )

Dimana :

K= Keliling

a = Panjang Alas

t = Tinggi

Hitung Keliling Jajar Genjang
